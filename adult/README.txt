Paper: https://arxiv.org/abs/2402.04382

<dataset_name>Folders Contain 2 files each: 
	1) <dataset_name>_FOLD-SE.pl: CFGS code using FOLD-SE Rules
	2) <dataset_name>_RIPPER.pl: CFGS code using RIPPER Rules


To run on s(CASP) , launch s(CASP) and run the following query (example used is the adult dataset):
	1) scasp adult_FOLD-SE.pl --prev_forall -s0

	2) scasp adult_RIPPER.pl --prev_forall -s0
